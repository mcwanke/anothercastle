# AnotherCastle

The **Capture** layer of BlackHole. A suite of client applications for silent URL routing and optional ingestion to Toadstool.

**Primary function:** Route URLs silently to the correct browser based on domain rules.  
**Secondary function:** Optionally save URLs and text selections to Toadstool.

2. Read **../blackhole-private/CLAUDE_SHARED.md** for shared working agreements
